# Yaro-Smart-Farm-Simulator
(Yaro Smart Farm Simulator) is an Intermediate project use in practising OOP 
Smart Catfish Farm Simulator

A Python OOP project that simulates fish farming operations.

Features:
- Tank management
- Fish batch management
- Biomass calculation
- Feeding simulation
- Water quality management
- Daily reporting

Built while learning Python Object-Oriented Programming.
